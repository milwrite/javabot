üìñ FEATURE DESCRIPTION

BUSHWICK is a JavaScript feature that configures a Leaflet map to automatically zoom in on the Bushwick neighborhood in Brooklyn, New York.

COORDINATES:
Latitude: 40.6942
Longitude: -73.9194
Zoom Level: 14

üéÆ HOW TO USE

STEP 1:
Include Leaflet CSS and JS libraries in your HTML
STEP 2:
Load the bushwick.js script file
STEP 3:
Create a div element with id="map"
STEP 4:
Initialize the map and call zoom function

üíª CODE EXAMPLE

<!-- HTML -->
<div id="map" style="height: 500px;"></div>

<!-- JavaScript -->
<script src="bushwick.js"></script>
<script>
  // Initialize map
  var map = L.map('map').setView([40.6942, -73.9194], 14);

  // Add tile layer
  L.tileLayer('https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png', {
    attribution: '¬© OpenStreetMap'
  }).addTo(map);

  // Add marker for Bushwick
  L.marker([40.6942, -73.9194]).addTo(map)
    .bindPopup('Welcome to Bushwick!');
</script>
